## Files to Download

Download the following datasets from the [GeoNutrition](https://github.com/rmlark/GeoNutrition) repository and organize them into country-specific subfolders.

### Ethiopia Subfolder (`Ethiopia/`)

Download and place the following files inside the `Ethiopia/` folder:

* [Ethiopia_map.csv](https://github.com/rmlark/GeoNutrition/blob/main/Geostatistics_and_mapping/Ethiopia_map.csv)
* [Ethiopia_data.xlsx](https://github.com/rmlark/GeoNutrition/blob/main/Soil_Crop_comparisons/Ethiopia/Ethiopia_data.xlsx)
* [GeoN_Ethiopia_pH_OCdata.xlsx](https://github.com/rmlark/GeoNutrition/blob/main/Soil_Crop_comparisons/Ethiopia/GeoN_Ethiopia_pH_OCdata.xlsx)

### Malawi Subfolder (`Malawi/`)

Download and place the following files inside the `Malawi/` folder:

* [Malawi_targets.csv](https://github.com/rmlark/GeoNutrition/blob/main/Geostatistics_and_mapping/Malawi_targets.csv)
* [Malawi_grain_soil.xlsx](https://github.com/rmlark/GeoNutrition/blob/main/Soil_Crop_comparisons/Malawi/Malawi_grain_soil.xlsx)

Data downloaded from the GeoNutrition repository remain subject to the original license terms.
